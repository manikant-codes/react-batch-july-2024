- Redux ek global state management library hai.
    - Global state yane aise state jinhe aap ko pure application me kafi jagah use/change karna padta hoo.
        - Jaise ke cart ka state, user ka state, etc.
    - Global state management yane inhi global states ko use/change karna.
    - Redux ek JS library hai jo JS applications (React, Angular, Vue, Node, JS Applicaiton) me aap ko aise hi cart jaise states manage karne deti hai.

- Ham do libraries ka use karenge ReduxToolkit aur ReactRedux.
    - Redux ko setup aur use karne ka process lengthy hai. Isi ko aasan banane ke liye ek aur library hai ReduxToolkit.
    - ReduxToolkit redux likhne ka naya tarika hai. Ye JS apps me redux ko use karna aasaan banata hai.
    - ReactRedux ye aap ko kuch hooks deta hai jis se react me redux use karna aasaan hojata hai.
    - ReduxToolkit redux ko aasaan banata hai aur ReactRedux redux ko react me use karna aasaan banata hai.
    - Redux -> ReduxToolkit -> ReactRedux -> React

- Redux Pattern
    - Yane Redux kaise kaam karta hai.
    - Is pattern me 3 main componets/chize hai:
        - Store:
            - Yane wo jagah jaha aap apne saare global states store karte/rakhte hoo.
            - Sare global states aap ek jagah pe rakhte hoo aur us jagah ka naam hai store.
            - Syntax ke hisab se baat kare to ye store ek object hai.
        - Reducer:
            - Reducer ek function hai jo aap ke liye store ke states me changes karta hai.
        - Action:
            - Action ek object hai jisme 2 properties hoti hain:
                - type
                - payload
            - Is action object ka use kar ke aap reducer ko batate hoo ke konse state me kya changes karne hai.

    - Aur 3 main principles/rules hai:
        - 1. Sare global state store me rakhe jayenge.
        - 2. Store ke states me directly changes nahi kiye jaa sakte hain. Jo bhi changes hain wo sirf reducer function kar sakta hain.
        - 3. Reducer function ko action dispatch kar ke batate hai ke konse state me kya changes karne hai.

- createAsyncThunk
    - Ek function hai jo aap ko thunk action creator / async action creator function return karta hai.
        - Action creator function yane ek aisa function jo aap ko action object return kare. Ye action object sync logic wale reducers ke liye hota hai.
        - Async action creators aise functions hai jo aap ko action object dete hai async logic wale reducers ke liye.

    - createAsyncThunk 2 arguments leta hai:
        1: Action type ki string
        2. Payload creator function
            - Is function ke andar se aap promise return karte hoo. Isi promise ka jo result hai wo payload banega.
            - Isi promise pe se wo promise ke lifecyle states (pending, fulfilled, rejected) ke actions dispatch karta hai.

App is dispatch karoge: dispatch(asynActionCreator())
    - Ye badle me 2 aur dispatch karega payload creator se jo promise return hua hai uske par se.
    - 1. pending wala
    - 2. fullfilled / rejected wala
